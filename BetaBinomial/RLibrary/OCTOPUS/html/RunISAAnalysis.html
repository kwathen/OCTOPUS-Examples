RunISAAnalysis

Description

Generic method called to run the ISA analysis. Because the decision is tied to the analysis, each RunISAAnalysis should contain the elements nGo, nNoGo, nPause as part ot the list retured.

Usage

RunISAAnalysis(cISAAnalysis, lDataAna, nISAAnalysisIndx,
  bIsFinalISAAnalysis, cRandomizer)
